The Full regression graph

To create an adjacency matrix for a RC graph we are going to use the package ggm. The main function used is makeMG (make a mixed graph).

library("ggm")
gfull <- makeMG(dg = DAG(F ~ M+E+A+U, M ~ U + A,  E ~ U + A), 
                bg = UG(~ M*E +  U*A))
gfull
